# muyu-mini

独立微信小程序项目（原生小程序），核心玩法：点击木鱼 -> 敲击反馈 + 功德累积。

## 目录

- `app.js`
- `app.json`
- `app.wxss`
- `sitemap.json`
- `pages/index/`
- `assets/audio/`

## 运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录指向：`apps/muyu-mini`。
4. AppID 可先用测试号。

## 功能

- 点击木鱼：音效池轮播播放 + 震动 + 点击缩放抖动。
- 动态加分浮层：支持并发，最多 10 个，超出回收最旧。
- 功德累计本地持久化。
- 设置项：声音、震动、动画开关、每次+1/+3/+9、音效切换、清零确认。
